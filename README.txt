== SEO Header Footer Code ==
A plugin to add custom code in header, footer, body for posts, pages or global

== Installation == 
Upload the plugin files to the /wp-content/plugins/seo-header-footer-code directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress
Use the Settings->SEO Header Footer screen to configure the plugin

 == Frequently Asked Questions == 
What does this plugin do?
This plugin add custom codes to website's <head>, <body>, <footer> tags. It makes SEO work easy.

 == How do I use this plugin? == 
To use this plugin, follow these steps:

Go to the plugin settings page (Settings->SEO header footer code)
Insert your header, body, footer code to show globally in your website.

Is this plugin customizable?
